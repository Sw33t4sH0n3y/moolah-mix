# Moolah M$x 

[moolah-M$x](https://payoutpitch-8fa758c596dd.herokuapp.com/)

**No Shady Splits. Just Solid Ownership.**
 A split sheet management app for musicians, producers, and songwiters who want to protect their work *before* the check clears.


 ## 🎯 The Problem

 You made a hit, now everyone wants a piece. But who owns what?

 Split sheets get lost. Handshake deal are forgotten. When the royalties roll in, choas..

 ## 💡 The Solution

 **Payout Pitch** Documents ownership, manage collaborators, and generate professional split sheets. All in one place.


 ## ✨ Features

 |  Feature  |  Description  |
 |-----------|---------------|
 |  🔐 **User Auth** | Secure sign-up/sign-in  |
 |  🎵 **Track management**  | Add, edit,delete your songs  |
 |  👥 **Collaborators**  | Add all contributors  |
 |  📊 **Split Sheets**  | Master + Publishing splits that equal 100%  |
 |  📄 **PDF Export**  | Download professional split sheets  |
 |  🔎 **Deezer Search**  | find tracks and auto-fill details  |
 |  🌙 **Dark/Light Mode**  | easy on eyes  |
 |  🎨 **Vinyl Record Design**  | warm colors, vinyl vibes. |

 ## 🛠️ Tech Stack 

 - **Backend:** Node.js, Express.js
 - **Database:** MongoDB, Mongoose
 - **Views:** EJS Templates
 - **Styling** Custom CSS
 - **Auth:** Express-Session
 - **PDF** PDFKit
 - **API** Deezer Public API
